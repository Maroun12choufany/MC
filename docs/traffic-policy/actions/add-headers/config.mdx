## Configuration Reference

This is the configuration reference details for this action. You can setup and configure this action through your Traffic Policy documents.

### Action Type

`add-headers`

### Configuration Fields

| Parameter | &nbsp;                    | Description                                                                                                                                              |
| --------- | ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `headers` | `map[string]string` | Map of header key to header value to be added. Minimum `1`, Maximum `10`. Supports [string interpolation](#templating-syntax) in header values. |

### Supported Directions

- `inbound`
- `outbound`

### Supported Schemes
- `https`
- `http`
